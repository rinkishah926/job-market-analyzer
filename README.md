# Job Market Intelligence Dashboard

This project analyzes real-world job postings data to identify:
- In-demand skills
- Salary trends
- Job role distribution
- Hiring locations

## Features
- Data cleaning and preprocessing (~60K records)
- Feature engineering from job descriptions
- Interactive dashboard using Streamlit and Plotly
- Role-based filtering and search functionality

## Tech Stack
- Python
- Pandas
- Plotly
- Streamlit
